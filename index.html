Shell


Sim, o seu velho amigo bash tem muitos truques.


  • Mas se é pra lembrar de uma coisa só, lembre do Ctrl-R.
  • Mas, a coisa mais importante do shell é o |, por que ...
  • Truques: pesquisa no histórico, !!, ^old^new, movimentação na linha,

O shell não para no /bin/sh

O poder do shell está na família de comandos unix.

  • find, xargs, grep, sed, awk ...
  • ... cut, sort, tr, wc e outros do coreutils.
  • A coisa mais importante do shell é o | (pipe)


ldapsearch uid|grep ^uid|cut -f2 -d" "|sort|xargs -L1 groups|sed 's/ : /;/'

O SSH faz isso?


  • Port forwarding com -L e -R (reverso!).
  • Ou X11 forwarding com o -X. Ou ainda agent forwarding.
  • Conexão SSH indireta (proxy), atalhos de hostnames, login seguro sem senhas, entre vários outros recursos.
